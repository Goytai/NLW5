<h2>🖥️ Getting started</h2>
Clone the project:

```bash
$ git clone https://github.com/Goytai/NLW5.git
```

Acess the folder

```bash
$ cd NLW5
```

Follow the steps below
```bash
# Install the dependencies
$ yarn

# Run local data server
$ yarn server

# Start the project
$ yarn dev
```
